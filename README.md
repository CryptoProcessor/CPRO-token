# README

CRPRO (Crypto Processor) smart contracts with base Hardhat and Foundry projects.

### What is this repository for?

This repository is a global reference point for the smart contracts related to the CPRO token and the Crypto Processor project as a whole

### Repository structure

#### contracts

The Solidity smart contracts and definitions

#### foundry

Test Foundry project for making/testing, deploying smart contracts

#### hardhat

Test HardHat project for making/testing, deploying smart contracts
